Support - That's My Font

Get help using That's My Font to create your custom handwriting font.

Getting Started

Requirements

  • iPad: The app is designed exclusively for iPad
  • Apple Pencil: Recommended for best drawing experience (drawing is restricted to Apple Pencil only)
  • iOS 13.0 or later: Required for app functionality

First Steps

  1. Open the app on your iPad
  2. You'll see the welcome screen, then the drawing interface
  3. Use Apple Pencil to draw each character (A-Z, a-z, 0-9)
  4. Each character has a 100px drawing box with a template guide
  5. Double-tap a character box to clear and redraw

Drawing Characters

Drawing Tools

  • Pencil: Standard drawing tool
  • Pen: Pressure-sensitive tool (varies stroke width based on pressure)
  • Brush Width: Adjustable from 0.5px to 5px
  • Smoothness Toggle: Enable for smoother curves

Drawing Tips

  • Draw naturally - the app captures your handwriting style
  • Use the template guide to help with character positioning
  • Double-tap to delete and redraw if needed
  • Only Apple Pencil works for drawing (finger touches are blocked)

Adjusting Characters

After drawing all basic characters (62/62), you can:

  1. Click "Adjust & Export" button
  2. Hover over characters to highlight them
  3. Click a character to select and adjust it
  4. Use position arrows to move characters (X and Y axis)
  5. Use the scale slider to resize characters (50% to 150%)

Optional Characters

After completing 62/62 basic characters, you can add optional character sets:

  • Punctuation: .,!?;:\'"-()[]{}
  • Special Characters: #@$%&*+=|\/~`^_<>
  • Symbols: ©®™°•…—–

Check the boxes to add character sets, then draw the selected characters below.

Previewing Your Font

  1. After adjusting characters, click "Continue to Preview"
  2. Type text in the preview area to test your font
  3. Default text: "The Quick brown fox jumps over the lazy dog"
  4. Preview updates in real-time as you type
  5. Click "Back to Adjust" if you need to make changes

Exporting Your Font

Export Steps

  1. In the Preview screen, click "Export TTF Font"
  2. Enter a name for your font (letters, numbers, spaces, hyphens, underscores only)
  3. Click "Export Font"
  4. iOS Share sheet will open
  5. Choose where to save or share your font:
    • Save to Files app
    • Share via AirDrop
    • Send via email
    • Save to iCloud Drive

Using Your Font

  • On Mac: Double-click the TTF file to install via Font Book
  • On Windows: Right-click TTF file → Install
  • On iOS/iPadOS: Use a font management app or install via Settings
  • Use your font in any app that supports custom fonts (Pages, Word, design apps, etc.)

Common Issues

Can't Draw with Finger

The app only accepts Apple Pencil input for drawing. This prevents accidental touches while scrolling. Use Apple Pencil to draw characters.

Font Not Appearing After Export

Check the Files app or the location you chose in the Share sheet. The font file should be saved there. On Mac, double-click the TTF file to install it.

Characters Look Too Small/Large

Use the adjustment panel to fine-tune character scale and position. Select a character and use the scale slider to resize it.

Can't Scroll in Adjustment Panel

The adjustment panel supports both horizontal and vertical scrolling. Scroll bars should appear when content exceeds the visible area.

Tips for Best Results

  • Draw characters consistently - try to maintain similar size and style
  • Use the template guides to help with alignment
  • Adjust characters after drawing to perfect spacing
  • Preview your font with different text to see how it looks
  • Test the exported font in different apps to ensure compatibility
