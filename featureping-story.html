What FeaturePing does

Release note in. In-product explainer out.

1Paste the release note

Raw engineering or PM notes — whatever you already have.

2AI drafts the user copy

Announcement, why it matters, tooltip, help text. Always editable, never auto-published.

3Point at the feature

A page and an element, e.g. button[data-action='export']. Verified before publish.

4User clicks "Show me"

The widget walks them to the feature and highlights it — across pages if needed.
