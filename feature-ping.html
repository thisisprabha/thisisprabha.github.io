What you're looking at

The widget is framework-agnostic vanilla JS — it works in React, Vue, Angular, Rails, Django, or plain HTML, because it only touches what every stack produces: the DOM.

In the full product, an admin pastes a raw release note, AI drafts the user-facing copy, the admin verifies the target and publishes — and the update reaches only the users whose plan/role match. Views, dismissals, and “Show me” clicks are counted so you know which releases actually landed.
